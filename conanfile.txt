# Conan v2 configuration for gleditor
# This project uses GNU Make and pkg-config. We rely on Conan to provide
# pkg-config files via the PkgConfigDeps generator and to set up build/run
# environments. No CMake is required.
#
# Usage (example):
#   - Install dependencies into an output folder (e.g., build/conan):
#       conan profile detect --force
#       conan install . -of build/conan -s build_type=Release --build=missing
#   - Activate build environment and pkg-config path for Make:
#       source build/conan/conanbuild.sh
#   - Build as usual:
#       make
#   - To run binaries that need Conan-provided runtime libs:
#       source build/conan/conanrun.sh
#       ./build/gleditor
#
# Notes:
#   - If your system already provides some libs (e.g., OpenGL/GLU), you can
#     remove or comment them in [requires]. When present, Conan will generate
#     consistent pkg-config files for them.
#   - Some packages may be platform-specific or versions may evolve on ConanCenter.
#     If a given version is unavailable for your platform, you can adjust the
#     version constraint or remove it to accept the latest.
